Claude + Antigravity 給老師的 AI 工作夥伴

AI 能做什麼

我是可以和你一起備課、設計活動、整理資料、撰寫溝通文字、檢查程式與製作教材的 AI 助手。 當我結合 Antigravity，就不只是聊天視窗，而是能直接在你的專案資料夾裡讀檔、改檔、整理內容、建立網頁，陪你把想法做成真正可用的東西。

我是誰，結合 Antigravity 後能做什麼

你可以把我想成「懂教學語境的文字夥伴」加上「能操作本機工作區的製作助理」。 你提出目標，我負責拆解步驟、產出草稿、修改檔案、做成網頁或文件，並在完成後協助檢查。

思考與規劃

把模糊需求整理成可執行方案

例如：你說「下週要教光合作用」，我可以幫你排 45 分鐘教案、提問流程、分組任務與收尾評量。

製作與修改

直接建立教學用檔案

例如：幫你在指定資料夾建立 HTML 教材、講義草稿、測驗題庫、通知單或班親會簡報大綱。

檢查與優化

協助你把內容變清楚

例如：檢查考卷題目是否對應學習目標、家長通知是否語氣得體、教材文字是否適合年級程度。

我能協助老師的日常工作

下面每一項都可以直接交給我做初稿，再由老師用專業判斷調整。AI 最適合處理重複、耗時、需要大量變化版本的工作。

出考卷與題庫 例：依照六年級數學「比與比值」出 20 題選擇、10 題填充、5 題應用題，附答案與解析。
差異化學習單 例：同一課國語文本，產出基礎、一般、挑戰三種閱讀理解學習單。
教案與課程流程 例：設計 40 分鐘自然課，包含引起動機、探究活動、學生發表與形成性評量。
教材製作 例：把課本重點整理成投影片架構、黑板板書、互動問答卡或網頁教材。
班級經營 例：為「學生常忘記帶作業」設計溫和但有結構的班級提醒制度與獎勵方式。
家長聯絡 例：協助撰寫親師溝通訊息，讓語氣清楚、尊重、具體，避免太生硬或太情緒化。
評語與回饋 例：依學生表現產出期末評語、作文回饋、學習歷程建議，並保留老師的溫度。
活動設計 例：規劃校外教學前導活動、安全提醒、任務單、回校後反思單。
行政文件 例：整理會議紀錄、研習心得、活動成果報告、計畫申請書初稿。
資料整理 例：把學生回饋、問卷文字、觀課紀錄整理成重點、趨勢與下一步建議。
跨語言支援 例：將給外籍家長的通知翻成英文、越南文或印尼文，並保持禮貌與易懂。
數位工具與網頁 例：做一個班級公告頁、互動小測驗、閱讀任務頁，或整理班級活動照片說明。

老師最該裝的 Antigravity 插件 / 擴充功能

以下推薦以「降低備課成本、提升文件品質、方便展示教材」為核心。若 Marketplace 名稱略有差異，可以搜尋相同功能類型。

Live Server

最適合做 HTML 教材或班級網頁。修改檔案後瀏覽器會快速更新，老師可以一邊改、一邊看學生實際會看到的畫面。

Markdown All in One

適合寫講義、會議紀錄、教案與閱讀單。Markdown 乾淨、不容易跑版，也方便之後轉成 PDF 或網頁。

Prettier

自動整理網頁與教材檔案格式。老師不必手動對齊縮排，檔案會更整齊，也比較方便 AI 繼續協助修改。

Draw.io / diagrams.net

適合畫流程圖、概念圖、班級座位表、教學架構圖。很多抽象概念用圖呈現，學生更容易理解。

PDF Viewer

方便在 Antigravity 裡直接查看教材、試卷、學習單 PDF，不用一直切換視窗，整理資料時很省心。

Python / Jupyter

如果老師要整理成績、問卷或統計資料，Python 與 Jupyter 可以把表格轉成圖表、摘要與分析報告。

小提醒：插件不必一次裝滿。先裝「Live Server、Markdown、Prettier」三個，通常就能處理大多數教材與文件製作。

怎麼跟我說話，回答會最好

把我當成會追問、會整理、會動手做的助教。你給我越多教學現場限制，我越能產出接近可直接使用的內容。

Do：說清楚對象、年級與時間

例如：「幫我設計四年級 40 分鐘國語課，學生閱讀程度中等，主題是人物描寫。」

Do：給我範例或格式

例如：「請照這份通知單的語氣，改寫成家長比較容易接受的版本。」

Do：直接說你要的成品

例如：「我要一份可列印的 A4 學習單，包含題目、作答區、教師備註。」

Don't：只說「幫我做教材」

太籠統會讓結果漂浮。最好補上科目、單元、年級、學生狀況、想達成的學習目標。

Don't：把 AI 當最後答案

考題正確性、學生個資、特殊生需求、家長溝通語氣，都需要老師最後確認。AI 是加速器，不是專業判斷的替代品。
